# Generative DDD: Scaling Domain-Driven Design for Solo Entrepreneurs

This repository contains the source for the Generative DDD book. Our method is tailored for solo entrepreneurs to build software faster by focusing on domain understanding without the need for large teams of experts.

## Project Structure

- `/book`: Core content organized into chapters.
- `/experiments`: Parallel exploration of multiple Generative DDD ideas.
- `/assets`: Images and diagrams.
- `/examples`: Prototypes and domain models.
- `/scripts`: Build and automation scripts.

## Getting Started

This book is built using [HonKit](https://github.com/honkit/honkit).

To build the book locally:
1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`
3. Build for production: `npm run build`
