# Contributing to Generative DDD

We welcome contributions from soloentrepreneurs, technical experts, and anyone interested in the future of DDD!

## How to contribute

1. **Suggest a Chapter or Experiment:** Open an issue to discuss new ideas.
2. **Submit a Draft:** If you have an idea for a chapter, feel free to submit a pull request with an initial draft.
3. **Refine Existing Content:** Help us audit traditional methods or explore modern trends.

## Tools

We use [HonKit](https://github.com/honkit/honkit) for the book generation.
- Edit files in the `/book` directory using Markdown.
- Update `book/SUMMARY.md` if you add a new chapter.
